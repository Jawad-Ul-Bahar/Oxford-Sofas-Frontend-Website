## 🌟 Overview

**Oxford Sofa** is a modern, responsive frontend furniture website showcasing elegant sofas and home furniture. Built with pure HTML, CSS, JavaScript, and Bootstrap, this project demonstrates modern web development practices while providing an engaging user experience for furniture enthusiasts and potential customers.

Perfect for:
- 🎯 Frontend development portfolios
- 🛋️ Furniture business websites
- 📚 Learning responsive web design
- 🚀 Quick deployment projects

## 🚀 Installation

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Quick Start

1. **Clone the repository:**
```bash
git clone https://github.com/Jawad-Ul-Bahar/oxford-sofa.git
cd oxford-sofa
```

2. **Open the project:**
   - **Option 1:** Open `index.html` directly in your browser
   - **Option 2:** Use a local server:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js (if you have live-server installed)
     live-server
     ```

3. **Access the website:**
   - Direct: Open `index.html` in browser
   - Local server: Visit `http://localhost:8000`
